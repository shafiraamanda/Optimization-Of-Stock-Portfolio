# 📊 Optimization of Stock Portfolio

This project focuses on building an optimized stock portfolio using a combination of clustering and financial risk analysis methods.

## 🚀 Objectives
- To group stocks using clustering techniques (Mean Shift)
- To optimize portfolio allocation using Mean-Variance Optimization
- To measure investment risk using Value at Risk (VaR)

## 📂 Dataset
- Stock price data from 2023 to 2025
- Stored in: `Stocks_Data_2023_2025.xlsx`

## ⚙️ Methods Used
- Mean Shift Clustering
- Mean-Variance Optimization (Markowitz Model)
- Value at Risk (VaR)

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib

## 📈 Output
- Clustered stock groups
- Optimal portfolio allocation
- Risk estimation using VaR

## 📌 Notes
This project is part of a data analysis and financial modeling study, focusing on portfolio optimization techniques.
